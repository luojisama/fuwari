# bot插件使用方法
通用获取帮助指令：/help [插件名]

| 插件名称                       | 功能           | 使用方法                                                                                                                                                                                     | 是否需要@ | 备注  |
| :------------------------- | :----------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :---- | --- |
| JMComic插件                  | 查询jm本子       | `jm下载 [jm号]` :下载指定jm号的本子<br>`jm查询 [jm号]` :查询指定jm号的本子<br>`jm搜索 [关键词]` :搜索包含关键词的本子                                                                                                         | 否     | 仅私聊 |
| PixivBot                   | 查询pixiv图片    | `看看榜<范围>`：查看pixiv榜单<br>`来张图`：从推荐插画随机抽选一张插画<br>`来张<关键字>图`：搜索关键字，从搜索结果随机抽选一张插画<br>`来张<用户>老师的图`：搜索画师，从该画师的插画列表里随机抽选一张插画<br>`看看图<插画ID>`：查看id为<插画ID>的插画<br>`还要`：重复上一次请求<br>`不够色`：获取上一张插画的相关推荐 | 否     |     |
| YetAnotherPicSearch        | 搜索图片出处       | 发送`/搜图 -h`查看详细帮助                                                                                                                                                                         | 否     |     |

| 三角洲助手                      | 三角洲相关功能      | `/三角洲帮助`查看详细帮助                                                                                                                                                                           | 否     |     |

| 天气查询                       | 查询天气         | `/天气 地名`                                                                                                                                                                                 | 否     |     |

| 蔚蓝档案今日运势                   | 获取今日运势       | `今日运势`                                                                                                                                                                                   | 否     |     |
